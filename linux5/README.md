# Linux/5 - XP Edition

A nostalgic Windows XP-themed browser experience with a friendly black cat companion!

## Features

- **Classic XP Luna Theme**: Authentic Windows XP blue gradient desktop and taskbar
- **Interactive Desktop**: Draggable icons and windows with classic XP styling
- **Start Menu**: Click the green Start button to access programs and options
- **Working Clock**: Real-time clock in the system tray
- **Black Cat Pet**: Your friendly desktop companion inspired by Bonzi Buddy
  - Click the cat for random messages
  - Drag the cat anywhere on the screen
  - Watch it react to your mouse movements
  - Enjoy idle animations (blinking, tail wagging)

## How to Run

Simply open `index.html` in any modern web browser:

1. **Using a web browser directly:**
   - Navigate to the `linux5` folder
   - Double-click `index.html`
   - Or right-click and select "Open with Browser"

2. **Using a local web server (recommended):**
   ```bash
   cd linux5
   python3 -m http.server 8080
   ```
   Then visit `http://localhost:8080` in your browser

## Usage Tips

- **Double-click** desktop icons to open windows
- **Click and drag** window title bars to move windows
- **Click** the minimize, maximize, or close buttons on windows
- **Click** the Start button to open the Start menu
- **Click** the cat for friendly messages and tips
- **Drag** the cat to move it around the screen
- **Wait** and the cat will occasionally speak to you automatically

## Built With

- Pure HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript (no frameworks!)
